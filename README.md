# 🎧 我的音源

为 LX Music 提供**自用会员音源**，用你自己的网易云 / QQ 音乐会员登录态获取高清播放地址。

---

## 📦 安装（只需做一次）

1. 把 **本仓库的全部文件** 复制到洛雪安装目录下（建议单独放一个文件夹，如 `我的音源/`）
2. 右键 **`首次使用安装-右键用PowerShell运行.ps1`** → 使用 PowerShell 运行（需要先装好 [Node.js](https://nodejs.org)）
3. 安装完成后双击 **`启动我的LX.vbs`**

> 前置条件：电脑已安装 Node.js。没装的话去 [nodejs.org](https://nodejs.org) 下载 LTS 版。

---

## 🔑 登录（首次启动时）

启动 VBS 后，如果检测到未登录，会自动在浏览器打开 **http://127.0.0.1:3000**：

- **网易云**：用网易云音乐 App 扫码
- **QQ 音乐**：浏览器登录 [y.qq.com](https://y.qq.com) → F12 → Application → Cookies → 复制粘贴

登录后 Cookie 保存在本地，下次启动就**不用再登录**了。

---

## 🎵 导入音源到洛雪

1. 洛雪 → **设置** → **基本设置** → **音乐来源**
2. **自定义源管理** → **导入**
3. 选择 `我的会员音源.js`
4. 在列表中启用「网易云(我的会员)」和「QQ音乐(我的会员)」

---

## 🚀 日常使用

以后每次听歌，双击 **`启动我的LX.vbs`** 就行：
- 已登录 → 直接打开洛雪
- 未登录 → 弹出登录页 → 登录完自动开洛雪
- 关掉洛雪 → 后端自动停止

---

## ⚙️ 补充说明

- 后端只监听 `127.0.0.1:3000`，仅本机可访问
- Cookie 保存在本地，不上传到任何服务器
- 仅供个人自有账号播放辅助使用
- 升级洛雪版本时，只需把新版放在同一目录，VBS 会自动识别
