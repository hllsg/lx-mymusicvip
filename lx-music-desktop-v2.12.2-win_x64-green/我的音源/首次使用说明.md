# 🎧 我的音源 — 使用说明

为 LX Music（洛雪音乐）提供**自用会员音源**，用你自己的网易云 / QQ 音乐会员登录态获取高清播放地址。

---

## 🆕 首次使用（别人拿到后怎么用）

1. 把 `我的音源/` 整个文件夹复制到你的**洛雪安装目录**下
2. 双击 **`首次使用安装.bat`** 自动安装依赖（需要 Node.js）
3. 安装完成后，双击 **`启动我的LX.vbs`** 即可

> 就这么简单，不需要改任何配置。

---

## 🚀 日常使用

双击 **`启动我的LX.vbs`** 即可，脚本会自动：
1. 静默启动后端服务
2. 检测登录状态
   - ✅ **已登录** → 直接打开洛雪
   - ❌ **未登录** → 弹出登录页面 → 处理完再开洛雪
3. 关闭洛雪时自动停止后端

---

## 🔑 登录方法

### 网易云音乐（扫码）

后端启动后访问 **http://127.0.0.1:3000**，点击「网易云音乐」标签页，用网易云音乐 App 扫描二维码即可。

### QQ 音乐（Cookie 导入）

1. 浏览器打开 [y.qq.com](https://y.qq.com) 并登录
2. 按 `F12` → `Application` → `Cookies` → `y.qq.com`
3. 复制所有 Cookie，粘贴到控制台「QQ 音乐」标签页保存

---

## 🎵 导入音源到洛雪

1. 打开洛雪 → **设置** → **基本设置** → **音乐来源**
2. 点击 **自定义源管理** → **导入**
3. 选择 `我的会员音源.js`
4. 在音乐来源列表中启用「网易云(我的会员)」和「QQ音乐(我的会员)」

---

## 📁 目录结构

```
lx-music-desktop-v2.12.2-win_x64-green/
├── lx-music-desktop.exe    洛雪主程序
├── 我的音源/                自定义音源（本目录）
│   ├── server.js           后端 API 服务
│   ├── 我的会员音源.js      音源脚本（导入洛雪）
│   ├── 启动我的LX.vbs       一键启动
│   ├── 启动.bat             手动启动后端
│   ├── public/index.html    Web 控制台
│   └── scripts/            其他音源脚本
└── ...
```

所有文件集成在洛雪应用文件夹内，无需额外目录。

---

## ⚙️ 技术说明

- 后端监听 `127.0.0.1:3000`，仅本机可访问
- Cookie 保存在本地 `.netease-cookie` / `.qq-cookie` 文件，不上传
- 所有请求通过你的会员登录态代理，不绕过任何付费机制
- 仅供个人自有账号播放辅助使用

---

## 🆙 洛雪版本升级

升级洛雪时，只需将新版文件夹放到同一目录下即可，VBS 会自动识别新版 exe，无需修改任何配置。
